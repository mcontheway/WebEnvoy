## 收敛结论

2. 候选能力描述必须与最小能力壳分层：

   - `FR-0017` 负责候选能力元数据与统一描述
3. 验证/重放/可信判断应拆到下游 FR，而不是和候选能力描述混在一起。
4. 读 / 写 / 下载必须在模型层统一表达，即便下载实现优先级稍后。
